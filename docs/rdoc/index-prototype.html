ThingFish

ThingFish is a highly-accessible network store for cataloging large files and their associated metadata.

  • ThingFish — Toplevel namespace module
  • ThingFish::AbstractClass — Adds abstract class helpers to a class.
  • ThingFish::AbstractClass::ClassMethods — Methods to be added to including classes
  • ThingFish::Client — The thingfish client library
  • ThingFish::Config — The configuration reader/writer class for ThingFish::Daemon.
  • ThingFish::Config::ConfigStruct — Hash-wrapper that allows struct-like accessor calls on nested hashes.
  • ThingFish::ConfigError — Error in configuration file
  • ThingFish::Constants — (no description)
  • ThingFish::Constants::HTTP — HTTP status and result constants
  • ThingFish::Constants::Patterns — (no description)
  • ThingFish::Daemon — ThingFish daemon class
  • ThingFish::DefaultHandler — The default top-level handler for the thingfish daemon
  • ThingFish::Error — General ThingFish exception class
  • ThingFish::FileStore — Base class for ThingFish FileStore plugins
  • ThingFish::FileStoreError — Error in a filestore plugin
  • ThingFish::FileStoreQuotaError — Some action could not be completed because it would make the filestore exceed its allotted quota.
  • ThingFish::Filter — Base class for ThingFish Filter plugins
  • ThingFish::Handler — Base class for ThingFish Handler plugins
  • ThingFish::Loggable — Add logging to a ThingFish class
  • ThingFish::MemoryFileStore — An in-memory filestore plugin for ThingFish
  • ThingFish::MemoryMetaStore — In-memory metadata store
  • ThingFish::MetaStore — Base class for ThingFish MetaStore plugins
  • ThingFish::MetaStore::ResourceProxy — Proxy class for metadata operations for a given resource.
  • ThingFish::MetadataHandler — The default top-level handler for the thingfish daemon
  • ThingFish::MultipartMimeParser — A class for parsing multipart mime documents from a stream.

  • ThingFish::PluginError — Error in a plugin
  • ThingFish::Request — Request wrapper class

  • ThingFish::Resource — Resource wrapper class
  • ThingFish::ResourceLoader — Adds some methods that can be used to load content from files in a resources directory.
  • ThingFish::Response — Response wrapper class
  • ThingFish::StaticResourcesHandler — Add the ability to service static content from a ThingFish::Handler‘s resource directory
  • ThingFish::StaticResourcesHandler::ClassMethods — Methods installed in including classes
  • ThingFish::Table — A case-insensitive multivalue hash that allows access via Symbol or String.
